📖 关于这个博客

这是一个由 Hexo 搭建、使用 Fluid 主题、托管在 GitHub Pages 的个人博客。

📦 这里有什么？

博客里主要收录了我折腾的一些项目：

  • 🎮 小游戏：1024、井字棋、恶魔轮盘等
  • 🛠️ 实用工具：EasySearch、MySub、RSS2IMG 等
  • 🧩 Minecraft 模组/数据包：各种好玩的小模组
  • 🔧 Scratch 扩展：自制的扩展和教程
  • 📚 技术笔记：一些踩坑记录和开发心得
  • 等等

✨ 特点

  • ✅ 所有文章都是 Markdown 格式
  • ✅ 源码完全公开在 GitHub 上
  • ✅ 自动部署，每次推送自动更新
  • ❌ 没有广告（除非我哪天穷疯了）
